أنواع الثغرات الشائعة

  • SQL Injection: إدخال أوامر SQL في الحقول غير المؤمنة.
  • XSS: إدخال أكواد JavaScript في صفحات الويب.
  • CSRF: تنفيذ أوامر غير مصرح بها باسم الضحية.
  • IDOR: الوصول إلى معلومات مستخدم آخر عن طريق تغيير ID بالرابط.
