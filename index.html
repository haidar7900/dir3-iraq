أنواع الثغرات في المواقع

  

    SQL Injection

    تُستغل لتنفيذ استعلامات ضارة على قاعدة البيانات عبر مُدخلات غير مفلترة.
  

  

    XSS

    تسمح بحقن سكريبت داخل الصفحة لسرقة الجلسات أو تشويه المحتوى.
  

  

    IDOR

    وصول غير مصرح به للبيانات عبر التلاعب بالمعرّفات.
  

  

    CSRF

    تنفيذ أوامر غير مرئية باسم المستخدم دون علمه.
